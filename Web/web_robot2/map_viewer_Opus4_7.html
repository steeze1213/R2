export TURTLEBOT3_MODEL=waffle_pi
ros2 launch turtlebot3_gazebo turtlebot3_world.launch.py

ros2 launch rosbridge_server rosbridge_websocket_launch.xml
  • W/S: 전진/후진
  • A/D: 좌/우 회전
  • Space: 정지
  • D-pad 버튼 클릭 / 길게 누르기 동일 동작
  • 키를 떼거나 버튼 밖으로 나가면 자동 정지
  • 탭/창 비활성화 시 안전 정지 발행
  • 맵 색: 진한 회색=장애물, 흰색=자유 공간, 회색=미탐색
  • 마커: 초록 화살표가 현재 로봇 위치와 헤딩
  • 좌표계: 화면 Y는 ROS Y를 뒤집어 표시 (ros2djs 관례)
  • θ는 도(°) 단위
  • 연결 안 됨: ros2 node list | grep rosbridge, ss -tnlp | grep 9090
  • 맵 안 보임: ros2 topic hz /map, ros2 topic echo /map --once
  • 조종 안 됨: ros2 topic echo /cmd_vel
  • file://로 열어 차단 시: python3 -m http.server 8000 후 http://localhost:8000/파일명
